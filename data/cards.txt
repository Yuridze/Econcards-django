
# Экономические обозначения (пример)
Фиксированные издержки -> FC (fixed costs)
Переменные издержки -> VC (variable costs)
Совокупные издержки -> TC (total costs)
Средние издержки -> AC (average costs)
Предельные издержки -> MC (marginal costs)
Выручка -> TR (total revenue)
Предельная выручка -> MR (marginal revenue)
Прибыль -> π (profit)
Спрос -> D (demand)
Предложение -> S (supply)
Равновесная цена -> P* (equilibrium price)
Количество -> Q (quantity)
Цена -> P (price)
Эластичность спроса -> Ed (price elasticity of demand)
Эластичность предложения -> Es (price elasticity of supply)
Валовое внутреннее продукт -> GDP (gross domestic product)
Инфляция -> CPI (consumer price index, proxy)
Ставка процента -> r (interest rate)
Капитал -> K (capital)
Труд -> L (labor)
